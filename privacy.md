# Privacy Policy – BeatPulse AI

BeatPulse AI does not collect or store personal user data outside of Discord’s normal operation.

Data that may be processed:
- Discord user IDs (for interaction handling)
- Messages sent to the bot (for AI responses)
- Temporary session memory (not stored permanently)

We do not:
- Sell user data
- Share data with third parties
- Store sensitive personal information

Music playback uses third-party services such as YouTube.

If you have concerns, contact: your@email.com
